QuickPilot 경력자용 가이드

운영은 손에 익으신 분용. 인성 순정 + 통합콜 + 카카오를 최고 성능으로 돌리기 위한 세팅값·기능만 추렸습니다.

⚙️ 설치 (갤럭시)

  • 위 가이드의 앱 다운로드 → APK 실행 → "이 출처 허용" 한 번 켜고 설치
  • Play Protect 경고는 무시하고 설치
  • 접근성이 회색이면 설정 → 앱 → QuickPilot → ⋮ → 제한된 설정 허용 후 접근성에서 켜기
  • 권한 4개: 접근성 / 다른 앱 위에 표시 / 알림 / 위치(항상 허용)

🛠️ 3엔진 세팅값

  • 통합콜 — 노출 15,000 / 차종 맞춤(오토바이·다마스·라보·1t 보이게, 안 되면 콜센터) / 글씨 16 촘촘하게
  • 통합콜 — 표기는 수수료 전 금액. QP 3만 = 통합콜 38,960원(순수익 환산)
  • 인성 — 인성1·2 각각 노출 2만 / 자동배차 경유·왕복 제외
  • 인성 — 환경 설정 → 배차 화면 모드 = '기존 배차 화면 모드' (신규 X). 이래야 QP가 정보를 정확히 읽음
  • 카카오 — 보임 20km · 잡힘 10km / 소리·진동 끄고 접어두기
  • ⚠ 통합콜은 지하·통신 끊김 후 차종이 5t로 초기화될 수 있음 — 오더창 비면 톱니바퀴에서 확인

📱 화면 구성

  • 통합콜 위 1/3 + 인성 아래 2/3 분할 → 가운데 ··· 메뉴(가로 3개) 별표로 바탕화면 저장
  • 구도 깨지면 저장한 아이콘 한 번이면 복구
  • 화면은 항상 켜둠(세컨폰 권장) — 백그라운드면 자동배차 성능 급락

🚀 QP 자동배차

  • 10km · 3만 원 고정, '시작'만. 타깃은 3~5만 원대 콜
  • 시작 후 QP는 뒤에서 돌고, 화면엔 통합+인성 분할을 띄워 둠. QP 화면 볼 필요 없음
  • 인성 좌하단 빨간 '정지' = 연결됨(가끔 안 맞으면 직접)
  • 통합콜 팝업 = 확정·상세 한 터치 / 카카오 조건 맞으면 빨간 테두리 → 수락만
  • 시작/정지 길게 = 확장 레이더(20km·5만, 통합콜만 거리 무관 인식)
  • 먼저 배차된 콜은 못 뺏어옴 — 새치기 구조상 불가

🎯 콜 판단 (토스트)

  • 토스트 = 정리된 주소 · 거리 · 방위표 · km당 단가 색
  • 방위표: 출발지=현 위치→출발 방향, 도착지=출발→도착 방향
  • 기준은 시간 효율(확정→픽업→완료 대비 금액), 적정 시간당 2.5~3.5만
  • 단가 색: 빨강 나쁨 → 초록 중간 → 파랑 최상 / 좋은 퀵사는 하트
  • 인성은 적요란이 분할에서도 보여 토스트 안 띄움 → 상세창 직접 확인

⚠️ 페널티 · 동선

  • 인성 취소 하루 5회(인성1·2 각각) 절대 넘기지 마라 — 잡으면 간다
  • 멈추지 마라. 서울·경기 밖은 찍고 바로 빠져나오기
  • 목적지·제외지 설정 X / 서울은 오후 4시 전 탈출
  • 가는 길 릴레이로 빠져나올 콜, 못 잡으면 던지고 수도권 방향 복귀
