# QuickPilot Landing

큐피 베타 테스터 모집 랜딩 페이지.

## 배포

GitHub Pages — `pnomade12-lgtm.github.io/quickpilot-landing`

## 구조

단일 HTML 파일. 모바일 우선, 메뉴 없는 long-scroll.

- Hero: 슬로건 (원문 보존 의무)
- 빌드업 스토리: 디렉터 직접 작성
- 어디까지 왔는가: 디렉터 직접 작성
- Manifesto: 원문 보존 의무
- 큰 그림: 디렉터 직접 작성
- 데모: 인터랙티브 MVP 자리
- CTA: 베타 테스터 신청

## 원문 보존 룰

`feedback_qp_slogan_manifesto.md` 적용. 슬로건과 manifesto는 디렉터 본인 외 누구도 한 글자도 안 만짐.
